# GenAI ETL Pipeline 🚀

## 🛠️ Tecnologias
- **Python** & **Pandas**
- **LangChain** (Orquestração de LLM)
- **OpenAI GPT-3.5** (Transformação de dados)

## 🚀 Como rodar
1. Clone o repo: `git clone https://github.com`
2. Instale as dependências: `pip install -r requirements.txt`
3. Crie um arquivo `.env` e adicione sua `OPENAI_API_KEY`.
4. Execute: `python main.py`
